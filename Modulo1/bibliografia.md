# Bibliografía — Módulo 1

## Marcos de referencia

- **MITRE ATLAS** — https://atlas.mitre.org/ — matriz operativa, case studies, herramientas (Arsenal).
- **NIST AI 100-2 2nd Ed. (2025)** — "Adversarial Machine Learning: A Taxonomy and Terminology" — https://csrc.nist.gov/pubs/ai/100/2/e2025/final
- **NIST AI RMF 1.0** — https://www.nist.gov/itl/ai-risk-management-framework — base de Módulo 8.
- **OWASP Machine Learning Security Top 10** — https://owasp.org/www-project-machine-learning-security-top-10/
- **OWASP Top 10 for LLM Applications 2025** — https://owasp.org/www-project-top-10-for-large-language-model-applications/

## Threat modeling clásico (necesario para STRIDE-AI)

- **"Threat Modeling: Designing for Security"** — Adam Shostack (2014). El libro de referencia.
- **Microsoft SDL Threat Modeling** — https://www.microsoft.com/en-us/securityengineering/sdl/threatmodeling

## Papers y artículos clave

- Hu et al., "A Survey on AI Threat Modeling" (2024).
- Carlini et al., "Are Aligned Neural Networks Adversarially Aligned?" (2023).
- Shokri et al., "Membership Inference Attacks Against Machine Learning Models" (2017) — base de NIST privacy.
- Tramèr et al., "Stealing Machine Learning Models via Prediction APIs" (2016) — base de model extraction.

## Case studies para usar en clase

- **Microsoft Tay (2016)** — https://en.wikipedia.org/wiki/Tay_(bot)
- **PoisonGPT (Mithril Security, 2023)** — https://blog.mithrilsecurity.io/poisongpt-how-we-hid-a-lobotomized-llm-on-hugging-face/
- **Bing Sydney system prompt leak (2023)** — varios artículos en Ars Technica, The Verge.
- **DeepFake CFO scam Hong Kong (2024)** — caso de impersonación con IA generativa.

## Herramientas para acompañar este módulo

- **MITRE ATLAS Navigator** — explorador interactivo de la matriz.
- **OWASP Threat Dragon** — herramienta libre para diagramas + STRIDE.
- **Microsoft Threat Modeling Tool** — gratis, con plantillas STRIDE.

## Lecturas recomendadas (orden sugerido)

1. NIST AI 100-2 (las primeras 30 páginas — taxonomía).
2. MITRE ATLAS — recorrer las 14 tácticas y leer 5 case studies.
3. OWASP LLM Top 10 (resumen).
4. Shostack capítulo 4 (Approaches to threat modeling).
5. Carlini et al. — adversarial alignment (paper denso pero esencial).
